# UADE-Ecommerce-AppInteractivas
TPO - UADE - APLICACIONES INTERACTIVAS - Ecommerce (Grupo 2)

## 📝 Descripción del Proyecto
Este proyecto consiste en el desarrollo del **Backend** para una aplicación de E-Commerce, enfocada en la transformación digital de las ventas online. 
La solución implementa una arquitectura robusta para gestionar usuarios, categorias y el flujo completo de compra y venta de productos

Desarrollado para la materia **Aplicaciones Interactivas** en **UADE**, Primer Cuatrimestre 2026.

## 🚀 Requerimientos del Sistema

### 🔐 Gestión de Usuarios
* **Registro:** `/api/usuarios` (POST) para registrar usuarios.
* **Login:** `/api/auth/login` (POST) para autenticación mediante mail y contraseña.

### 📦 Catálogo de Productos
* **Listados:** Acceso a productos y categorías disponibles.

### 🛠️ Gestión de Productos (Vendedor)
* **Publicaciones:** Alta de productos, descripción y categoría.
* **Administración:** El creador del producto puede manejar el stock disponible.

## 🏗️ Especificaciones Técnicas
Para cumplir con los requisitos de entrega, el proyecto incluye:
* **Capa de Persistencia:** Implementada a partir del modelo de negocio entregado.
* **API Rest:** Desarrollada para acceder a la totalidad de la información, permitiendo consultas completas o filtradas.
